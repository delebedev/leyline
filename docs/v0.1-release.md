# v0.1 — First Public Release

The first version we advertise. Someone sees this project, clones it, plays a game. It works.

## Core set: Foundations (FDN)

FDN is the flagship set. All 517 cards have Forge scripts (100% coverage). A player should be able to build an FDN deck, play a game, and have every card work as advertised.

**Formats:**
- Constructed (bot match with FDN deck)
- Sealed (6-booster FDN pool → build → play)
- Draft (3-pack FDN draft vs bots → build → play)
- Brawl vs AI (stretch — Arena doesn't offer this, strong differentiator)

**Card mechanics in FDN:**
- 15 evergreen keywords (flying, trample, deathtouch, first strike, double strike, haste, flash, vigilance, reach, menace, hexproof, indestructible, defender, prowess, lifelink)
- Named mechanics: Raid, Landfall, Kicker, Flashback, Morbid, Surveil, Scry, Affinity, Changeling, Alliance, Crew, Fight, Mill
- 123 ETB trigger cards, 24 dies triggers, 19 attack triggers
- Tokens, +1/+1 counters, equipment, auras, planeswalkers

## Set-agnostic: the game has to feel right

These apply regardless of which set you play. Discovered through playtesting, not speculation.

**Must work:**
- Turn structure flows without unnecessary stops
- Combat: attack, block, damage — no stalls or wrong outcomes
- Combat VFX — correct attack/block animations (currently uses wrong animation, feels broken)
- Auto-pass reflected in UI — engine auto-passes must not leave client in limbo
- Targeting: spells and abilities target correctly, cancel works
- Mana: auto-tap, manual tap, treasures
- Stack: spells resolve LIFO, counters work, fizzle works
- Mulligan: London mulligan with correct hand size
- Game over: lethal damage and concede both show result screen
- Timer rope appears (even if generous — no Arena-style polish needed)

**Must not break:**
- No client crashes or soft-locks in normal FDN play
- No invisible cards or missing zones
- No stuck priority (player can always act or pass)

**Known gaps to assess (playtesting will tell):**
- Library search — some modes may not work
- Scry visual — annotation has wrong detail keys
- Phase indicator — player/AI sync broken
- Ward — partial
- Landfall/Raid/Morbid — turn-scoped trigger fidelity unclear
- Mana pool display — disabled
- Shuffle animation — suppressed (cosmetic)

## First-run experience

**README:**
- Clear "what this is" — one paragraph
- Prerequisites with versions
- Clone → build → run in under 5 commands
- "Start a game" walkthrough (launch Arena → connect → play)
- What to expect / known limitations

**Install (macOS + Windows):**
- `just bootstrap` works on a fresh clone
- TLS cert setup is scripted, not manual
- Card database sourced from local Arena install (no distribution)
- Windows: NSIS installer (launcher bundles jlink runtime)
- macOS: DMG installer (launcher bundles jlink runtime)

## Out of scope for v0.1

- Multiplayer (PvP)
- Best-of-three / sideboarding
- Sets beyond FDN (they may work via Forge, but not promised)
- Reconnect after disconnect
- Emotes
- Arena-style timer polish
- Manual phase stop configuration (unless playtesting shows it blocks normal play)
- Performance optimization
- Card art or cosmetics distribution
