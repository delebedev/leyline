# Scry — MTGA Client Log State Accumulator

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Parse MTGA Player.log GRE messages into accumulated game state, exposable via CLI and HTTP for consumption by leyline debug tooling and arena automation.

**Architecture:** Standalone Python tool in `bin/scry`. Tails Player.log, extracts GRE JSON blocks, applies Full/Diff state merge to maintain current game state per match. Two interfaces: CLI (`scry state`, `scry stream`) for unix composability, tiny HTTP server (`scry serve`) for debug server integration.

**Tech Stack:** Python 3.14, stdlib only (json, re, http.server, argparse). pytest for tests. No external deps.

---

## Meeting Notes (2026-03-08)

**Context:** Our debug tooling tracks Forge engine state and wire protocol exhaustively, but has zero positive signal from the client side. PlayerLogWatcher only captures exceptions. Meanwhile, every MTGA tracker in the ecosystem (17Lands, Untapped, MTGA Tool) parses Player.log GRE messages for full game state — this is a solved, well-documented log format.

**Decision:** Build our own focused tool rather than adapt existing OSS parsers (wrong languages — Go/JS/C#/Python, wrong scope — they track inventory/drafts/rank). Our scope: GRE GameStateMessages only, for state accumulation and divergence detection.

**Key scope decisions:**
- **Multi-game:** Track current game + keep last N completed games as history (option C). Reset on new ConnectResp.
- **Mid-session catch-up:** Scan backwards from EOF for last `GameStateType_Full`, accumulate forward from there (option A). Handles `scry` starting after game already in progress.
- **Output contract:** `scry state` prints JSON to stdout (CLI/script use). `scry serve --port 8091` runs tiny HTTP server for debug server integration (option A+B).
- **Language:** Python — matches arena tooling. Kotlin debug server consumes via HTTP.
- **No persistence** — in-memory accumulation only for now.

**Layering:**
- Layer 0 (Parser): Tail log, extract GRE JSON blocks, expose raw stream
- Layer 1 (Accumulator): Full/Diff merge → current state dict
- Layer 2 (Semantic): Zone tracking, annotation receipt, divergence vs server (future)

**Reference:** `~/src/mtg-lab` has a Go implementation of the same accumulator pattern (see `mtg/accumulator.go`, `mtg/gamestate.go`). Different goals but validates the Full/Diff merge approach.

## Card Name Resolution

Scry resolves grpId → card name using Arena's client SQLite database. This is the same DB backing `just card <grpId>`.

- **DB path:** `~/Library/Application Support/com.wizards.mtga/Downloads/Raw/Raw_CardDatabase_*.mtga` (Arena client's own SQLite DB, same as `just card` uses)
- **Module:** `cards.py` — `CardResolver` class, lazy-loaded, caches lookups
- **Usage:** CLI formatter uses it for human output (`--pretty` or TTY default). Models/accumulator stay numeric (grpIds) — no name coupling in the data layer.
- **JSON output** (piping/debug server): raw grpIds, no resolution
- **Human output** (TTY): `"Grizzly Bears (grpId=12345) moved to Battlefield"` style

```python
"""Game state models parsed from MTGA GRE messages."""

from __future__ import annotations

from dataclasses import dataclass, field


@dataclass
class TurnInfo:
    turn_number: int = 0
    phase: str = ""
    step: str = ""
    active_player: int = 0
    priority_player: int = 0
    decision_player: int = 0

    @classmethod
    def from_raw(cls, raw: dict) -> TurnInfo:
        return cls(
            turn_number=raw.get("turnNumber", 0),
            phase=raw.get("phase", ""),
            step=raw.get("step", ""),
            active_player=raw.get("activePlayer", 0),
            priority_player=raw.get("priorityPlayer", 0),
            decision_player=raw.get("decisionPlayer", 0),
        )


@dataclass
class Player:
    seat_number: int = 0
    life_total: int = 0
    max_hand_size: int = 7
    team_id: int = 0
    controller_seat_id: int = 0
    starting_life_total: int = 0
    status: str = ""
    pending_message_type: str = ""

    @classmethod
    def from_raw(cls, raw: dict) -> Player:
        return cls(
            seat_number=raw.get("systemSeatNumber", 0),
            life_total=raw.get("lifeTotal", 0),
            max_hand_size=raw.get("maxHandSize", 7),
            team_id=raw.get("teamId", 0),
            controller_seat_id=raw.get("controllerSeatId", 0),
            starting_life_total=raw.get("startingLifeTotal", 0),
            status=raw.get("status", ""),
            pending_message_type=raw.get("pendingMessageType", ""),
        )


@dataclass
class Zone:
    zone_id: int = 0
    type: str = ""
    visibility: str = ""
    owner_seat_id: int = 0
    object_ids: list[int] = field(default_factory=list)

    @classmethod
    def from_raw(cls, raw: dict) -> Zone:
        return cls(
            zone_id=raw.get("zoneId", 0),
            type=raw.get("type", ""),
            visibility=raw.get("visibility", ""),
            owner_seat_id=raw.get("ownerSeatId", 0),
            object_ids=raw.get("objectInstanceIds", []),
        )


@dataclass
class GameObject:
    instance_id: int = 0
    grp_id: int = 0
    type: str = ""
    zone_id: int = 0
    owner_seat_id: int = 0
    controller_seat_id: int = 0
    card_types: list[str] = field(default_factory=list)
    subtypes: list[str] = field(default_factory=list)
    colors: list[str] = field(default_factory=list)
    power: int | None = None
    toughness: int | None = None
    is_tapped: bool = False
    name: int = 0

    @classmethod
    def from_raw(cls, raw: dict) -> GameObject:
        power_raw = raw.get("power")
        toughness_raw = raw.get("toughness")
        return cls(
            instance_id=raw.get("instanceId", 0),
            grp_id=raw.get("grpId", 0),
            type=raw.get("type", ""),
            zone_id=raw.get("zoneId", 0),
            owner_seat_id=raw.get("ownerSeatId", 0),
            controller_seat_id=raw.get("controllerSeatId", 0),
            card_types=raw.get("cardTypes", []),
            subtypes=raw.get("subtypes", []),
            colors=raw.get("color", []),
            power=power_raw["value"] if power_raw else None,
            toughness=toughness_raw["value"] if toughness_raw else None,
            is_tapped=raw.get("isTapped", False),
            name=raw.get("name", 0),
        )


@dataclass
class GameState:
    game_state_id: int = 0
    type: str = ""
    prev_game_state_id: int | None = None
    turn_info: TurnInfo | None = None
    players: list[Player] = field(default_factory=list)
    zones: list[Zone] = field(default_factory=list)
    objects: dict[int, GameObject] = field(default_factory=dict)
    annotations: list[dict] = field(default_factory=list)
    actions: list[dict] = field(default_factory=list)
    game_info: dict | None = None
    diff_deleted_instance_ids: list[int] = field(default_factory=list)

    @property
    def is_full(self) -> bool:
        return self.type == "GameStateType_Full"

    @property
    def is_diff(self) -> bool:
        return self.type == "GameStateType_Diff"

    def zones_by_type(self, zone_type: str) -> list[Zone]:
        return [z for z in self.zones if z.type == zone_type]

    @classmethod
    def from_raw(cls, raw: dict) -> GameState:
        objects = {}
        for obj_raw in raw.get("gameObjects", []):
            obj = GameObject.from_raw(obj_raw)
            objects[obj.instance_id] = obj

        return cls(
            game_state_id=raw.get("gameStateId", 0),
            type=raw.get("type", ""),
            prev_game_state_id=raw.get("prevGameStateId"),
            turn_info=TurnInfo.from_raw(raw["turnInfo"]) if "turnInfo" in raw else None,
            players=[Player.from_raw(p) for p in raw.get("players", [])],
            zones=[Zone.from_raw(z) for z in raw.get("zones", [])],
            objects=objects,
            annotations=raw.get("annotations", []),
            actions=raw.get("actions", []),
            game_info=raw.get("gameInfo"),
            diff_deleted_instance_ids=raw.get("diffDeletedInstanceIds", []),
        )
```

```python
# bin/scry_lib/parser.py
"""Parse MTGA Player.log for GRE message blocks."""

from __future__ import annotations

import json
import re
from dataclasses import dataclass, field

# Match: [UnityCrossThreadLogger]08/03/2026 10:21:24: Match to <uuid>: GreToClientEvent
_GRE_HEADER = re.compile(
    r"\[UnityCrossThreadLogger\]"
    r"(\d{2}/\d{2}/\d{4} \d{2}:\d{2}:\d{2}): "
    r"Match to ([0-9a-f-]+): GreToClientEvent"
)


@dataclass
class GREBlock:
    """A single GreToClientEvent with its parsed messages."""
    messages: list[dict] = field(default_factory=list)
    timestamp: str | None = None
    match_id: str | None = None
    raw_json: dict | None = None

    @property
    def has_game_state(self) -> bool:
        return any(m.get("type") == "GREMessageType_GameStateMessage" for m in self.messages)

    def game_state_messages(self) -> list[dict]:
        return [m for m in self.messages if m.get("type") == "GREMessageType_GameStateMessage"]


def parse_gre_blocks(lines: list[str] | iter) -> iter:
    """Yield GREBlock from log lines. Expects raw Player.log format."""
    lines_iter = iter(lines)
    for line in lines_iter:
        line = line.rstrip()
        m = _GRE_HEADER.match(line)
        if not m:
            continue
        timestamp, match_id = m.group(1), m.group(2)

        # Next line should be JSON payload
        try:
            json_line = next(lines_iter).rstrip()
        except StopIteration:
            break

        try:
            data = json.loads(json_line)
        except json.JSONDecodeError:
            continue

        messages = (
            data.get("greToClientEvent", {}).get("greToClientMessages", [])
        )
        if messages:
            yield GREBlock(
                messages=messages,
                timestamp=timestamp,
                match_id=match_id,
                raw_json=data,
            )
```

```python
# bin/scry_lib/accumulator.py
"""Full/Diff state accumulator for MTGA game states."""

from __future__ import annotations

from collections import OrderedDict
from scry_lib.models import GameState, Player, Zone, GameObject, TurnInfo

MAX_HISTORY = 8


class Accumulator:
    """Maintains current game state by applying Full and Diff updates."""

    def __init__(self):
        self.current: GameState | None = None
        self._history: OrderedDict[int, GameState] = OrderedDict()

    def apply(self, state: GameState) -> None:
        if state.is_full:
            self.current = state
        elif self.current is None:
            # No base state — treat diff as full (best effort)
            self.current = state
        else:
            self.current = self._merge(self.current, state)

        self._history[self.current.game_state_id] = self.current
        while len(self._history) > MAX_HISTORY:
            self._history.popitem(last=False)

    def reset(self) -> None:
        self.current = None
        self._history.clear()

    def get_state(self, gsid: int) -> GameState | None:
        return self._history.get(gsid)

    def _merge(self, base: GameState, diff: GameState) -> GameState:
        # Turn info: diff wins if present
        turn_info = diff.turn_info if diff.turn_info else base.turn_info

        # Players: merge by seat number
        players = list(base.players)
        if diff.players:
            seat_map = {p.seat_number: i for i, p in enumerate(players)}
            for dp in diff.players:
                if dp.seat_number in seat_map:
                    players[seat_map[dp.seat_number]] = dp
                else:
                    players.append(dp)

        # Zones: diff zones replace base zones by zoneId
        zone_map = {z.zone_id: z for z in base.zones}
        for z in diff.zones:
            zone_map[z.zone_id] = z
        zones = sorted(zone_map.values(), key=lambda z: z.zone_id)

        # Objects: merge by instanceId, apply deletes
        objects = dict(base.objects)
        for iid, obj in diff.objects.items():
            if iid in objects:
                objects[iid] = self._merge_object(objects[iid], obj)
            else:
                objects[iid] = obj
        for iid in diff.diff_deleted_instance_ids:
            objects.pop(iid, None)

        # Game info: diff wins if present
        game_info = diff.game_info if diff.game_info else base.game_info

        return GameState(
            game_state_id=diff.game_state_id,
            type=diff.type,
            prev_game_state_id=diff.prev_game_state_id,
            turn_info=turn_info,
            players=players,
            zones=zones,
            objects=objects,
            annotations=diff.annotations,  # annotations are per-message, not cumulative
            actions=diff.actions,
            game_info=game_info,
        )

    def _merge_object(self, base: GameObject, diff: GameObject) -> GameObject:
        return GameObject(
            instance_id=diff.instance_id or base.instance_id,
            grp_id=diff.grp_id or base.grp_id,
            type=diff.type or base.type,
            zone_id=diff.zone_id or base.zone_id,
            owner_seat_id=diff.owner_seat_id or base.owner_seat_id,
            controller_seat_id=diff.controller_seat_id or base.controller_seat_id,
            card_types=diff.card_types or base.card_types,
            subtypes=diff.subtypes or base.subtypes,
            colors=diff.colors or base.colors,
            power=diff.power if diff.power is not None else base.power,
            toughness=diff.toughness if diff.toughness is not None else base.toughness,
            is_tapped=diff.is_tapped if diff.is_tapped else base.is_tapped,
            name=diff.name or base.name,
        )
```

```python
# bin/scry_lib/tail.py
"""File tailing with backward scan for mid-session catch-up."""

from __future__ import annotations

import os
import time
from pathlib import Path

_CHUNK_SIZE = 64 * 1024  # 64KB read chunks for backward scan


def scan_backward_for_full(path: Path) -> int | None:
    """Scan backward from EOF to find the byte offset of the last Full GSM.

    Returns the offset of the log header line preceding the Full GSM,
    or None if no Full state found.
    """
    size = path.stat().st_size
    if size == 0:
        return None

    last_full_offset = None

    with open(path, "rb") as f:
        # Scan forward (simpler, still fast for <100MB logs)
        # Player.log is typically 5-50MB
        offset = 0
        for line in f:
            decoded = line.decode("utf-8", errors="replace")
            if "GreToClientEvent" in decoded:
                header_offset = offset
            if '"GameStateType_Full"' in decoded:
                last_full_offset = header_offset
            offset += len(line)

    return last_full_offset


def tail_log(
    path: Path,
    follow: bool = False,
    start_offset: int = 0,
    poll_interval: float = 0.5,
) -> iter:
    """Yield lines from a log file, optionally following new writes.

    Args:
        path: Log file path.
        follow: If True, keep tailing for new lines (like tail -f).
        start_offset: Byte offset to start reading from.
        poll_interval: Seconds between polls in follow mode.
    """
    with open(path) as f:
        if start_offset > 0:
            f.seek(start_offset)

        while True:
            line = f.readline()
            if line:
                yield line
            elif follow:
                time.sleep(poll_interval)
                # Check for truncation (client relaunch)
                try:
                    if f.tell() > path.stat().st_size:
                        f.seek(0)
                except OSError:
                    break
            else:
                break
```

```python
# bin/scry_lib/tracker.py
"""Multi-game tracker with history."""

from __future__ import annotations

from dataclasses import dataclass, field
from scry_lib.accumulator import Accumulator
from scry_lib.models import GameState
from scry_lib.parser import GREBlock


@dataclass
class CompletedGame:
    match_id: str
    final_state: GameState | None


class GameTracker:
    """Tracks current game + keeps completed games as history."""

    def __init__(self, max_history: int = 5):
        self._accumulator = Accumulator()
        self._max_history = max_history
        self.current_match_id: str | None = None
        self.completed_games: list[CompletedGame] = []

    @property
    def current_state(self) -> GameState | None:
        return self._accumulator.current

    def feed(self, block: GREBlock) -> None:
        # Check for ConnectResp → new game
        has_connect = any(
            m.get("type") == "GREMessageType_ConnectResp"
            for m in block.messages
        )

        if has_connect and self.current_match_id is not None:
            # Archive current game
            self._archive_current()

        if block.match_id:
            self.current_match_id = block.match_id

        # Apply all game state messages
        for msg in block.game_state_messages():
            gsm = msg.get("gameStateMessage", {})
            if gsm:
                self._accumulator.apply(GameState.from_raw(gsm))

    def _archive_current(self) -> None:
        if self.current_match_id:
            self.completed_games.append(
                CompletedGame(
                    match_id=self.current_match_id,
                    final_state=self._accumulator.current,
                )
            )
            while len(self.completed_games) > self._max_history:
                self.completed_games.pop(0)
        self._accumulator.reset()
        self.current_match_id = None

    def to_dict(self) -> dict:
        state = self.current_state
        if state is None:
            return {"match_id": self.current_match_id, "state": None}

        return {
            "match_id": self.current_match_id,
            "game_state_id": state.game_state_id,
            "turn_info": {
                "turn_number": state.turn_info.turn_number,
                "phase": state.turn_info.phase,
                "step": state.turn_info.step,
                "active_player": state.turn_info.active_player,
                "priority_player": state.turn_info.priority_player,
            } if state.turn_info else None,
            "players": [
                {"seat": p.seat_number, "life": p.life_total}
                for p in state.players
            ],
            "zones": {
                z.type: {
                    "zone_id": z.zone_id,
                    "owner": z.owner_seat_id,
                    "objects": z.object_ids,
                }
                for z in state.zones
                if z.object_ids  # skip empty zones for brevity
            },
            "object_count": len(state.objects),
            "completed_games": len(self.completed_games),
        }
```

```python
# bin/scry_lib/cli.py
"""CLI interface for scry."""

from __future__ import annotations

import argparse
import json
import sys
from pathlib import Path

from scry_lib.parser import parse_gre_blocks
from scry_lib.tail import scan_backward_for_full, tail_log
from scry_lib.tracker import GameTracker

DEFAULT_LOG = Path.home() / "Library/Logs/Wizards of the Coast/MTGA/Player.log"


def cmd_state(args):
    log_path = Path(args.log)
    if not log_path.exists():
        json.dump({"error": f"log not found: {log_path}", "state": None}, sys.stdout)
        sys.stdout.write("\n")
        return

    # Catch-up: scan backward for last Full, replay forward
    start_offset = 0
    if args.catchup:
        offset = scan_backward_for_full(log_path)
        if offset is not None:
            start_offset = offset

    lines = list(tail_log(log_path, follow=False, start_offset=start_offset))
    tracker = GameTracker()
    for block in parse_gre_blocks(lines):
        tracker.feed(block)

    json.dump(tracker.to_dict(), sys.stdout, indent=2)
    sys.stdout.write("\n")


def cmd_stream(args):
    log_path = Path(args.log)
    if not log_path.exists():
        print(f"log not found: {log_path}", file=sys.stderr)
        sys.exit(1)

    follow = args.follow
    lines = tail_log(log_path, follow=follow)
    for block in parse_gre_blocks(lines):
        summary = {
            "match_id": block.match_id,
            "timestamp": block.timestamp,
            "message_types": [m.get("type", "") for m in block.messages],
            "gsids": [m.get("gameStateId") for m in block.messages if "gameStateId" in m],
        }
        json.dump(summary, sys.stdout)
        sys.stdout.write("\n")
        sys.stdout.flush()


def cmd_serve(args):
    from scry_lib.server import run_server
    run_server(log_path=Path(args.log), port=args.port)


def main():
    parser = argparse.ArgumentParser(prog="scry", description="MTGA Player.log state accumulator")
    parser.add_argument("--log", default=str(DEFAULT_LOG), help="Path to Player.log")
    sub = parser.add_subparsers(dest="command")

    state_p = sub.add_parser("state", help="Print accumulated game state as JSON")
    state_p.add_argument("--no-catchup", dest="catchup", action="store_false", default=True,
                         help="Don't scan backward for catch-up")

    stream_p = sub.add_parser("stream", help="Stream GRE blocks as JSONL")
    stream_p.add_argument("-f", "--follow", action="store_true", help="Follow log (like tail -f)")

    serve_p = sub.add_parser("serve", help="Run HTTP server")
    serve_p.add_argument("--port", type=int, default=8091, help="Port (default: 8091)")

    args = parser.parse_args()
    if args.command == "state":
        cmd_state(args)
    elif args.command == "stream":
        cmd_stream(args)
    elif args.command == "serve":
        cmd_serve(args)
    else:
        parser.print_help()
```
