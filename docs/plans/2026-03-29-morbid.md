# Morbid AbilityWordActive (#177)

## Context

Cackling Prowler, Needletooth Pack, and ~55 other Morbid cards in Forge never emit the
`AbilityWordActive` persistent annotation with `AbilityWordName="Morbid"`. The morbid
trigger itself works — Forge evaluates `Count$Morbid` and fires the end-step +1/+1 counter
ability correctly. The missing piece is the cosmetic pAnn that shows the client the morbid
condition badge.

**Issue:** #177
**Branch:** `feat/morbid-ability-word`
**Cards:** Cackling Prowler (grpId 93814), Needletooth Pack — all cards with Morbid keyword
**Card spec:** `docs/card-specs/cackling-prowler.md`
**Design spec:** `docs/specs/2026-03-29-morbid-ability-word-design.md`
**Sessions:** `2026-03-29_16-18-08`, `2026-03-29_16-32-23`

## Premise Verification

Wire shape confirmed from two sessions:

```
persistentAnnotations {
  types: AbilityWordActive
  affectorId: 1                    // player seat (NOT permanent iid, NOT ability iid)
  affectedIds: [323]               // all morbid permanent iids for that player
  details {
    key: "AbilityWordName"
    type: String
    valueString: "Morbid"
  }
}
```

Key differences from existing ability words (Threshold, Metalcraft, Delirium):

| Property | Threshold | Morbid |
|----------|-----------|--------|
| `affectorId` | permanent iid | seatId (1 or 2) |
| `affectedIds` | `[permanent iid]` | all morbid permanents for that player |
| `value` detail | present (GY count) | absent |
| `threshold` detail | present (7) | absent |
| Emission | one pAnn per permanent | one pAnn per player |
| Active condition | value >= threshold | any creature died this turn |

Fires reactively in the same diff as creature death (`ZoneTransfer(SBA_Damage)`) — not at
phase boundary. Our snapshot-compare pipeline naturally handles this: `buildFromGame()` runs
`AbilityWordScanner.scan()` after every diff, so if a creature just died, the next scan
sees morbid = true and emits the pAnn.

## Approach

Two emission modes in `AbilityWordScanner`:

**Per-permanent** (existing — Threshold, Metalcraft, Delirium): one pAnn per permanent,
`affectorId=permanentIid`, `affectedIds=[permanentIid]`, numeric value/threshold in details.

**Per-player** (new — Morbid, Raid): one pAnn per player, `affectorId=seatId`,
`affectedIds=[all morbid permanents for that player]`, boolean-only (no value/threshold).
Only emitted when condition is true.

Add two flags to `ConditionSpec`:
- `perPlayer: Boolean = false` — switches emission mode
- `booleanOnly: Boolean = false` — suppresses value/threshold from `AnnotationBuilder.abilityWordActive()`

The value lambda for Morbid needs a `Card` to call `CardUtil.getThisTurnEntered`. Change the
lambda type from `((Player) -> Int)?` to `((Player, Card) -> Int)?` — the second param is any
morbid card on the battlefield (used as the `src` arg Forge requires; any card in the game
works because the method uses `src.getGame()` internally).

private data class ConditionSpec(

### Task 2 — Per-player emission loop in `scan()`

**File:** `matchdoor/src/main/kotlin/leyline/game/AbilityWordScanner.kt`

After the existing per-permanent scan, add a second pass for `perPlayer` conditions.

The per-player pAnn has:
- `affectorId` = seatId (1 or 2 — from `bridge.getPlayer(SeatId(N)).seatNumber`)
- `affectedIds` = all morbid permanent iids for that player (collected from the battlefield scan)
- `instanceId` in `AbilityWordEntry` = seatId (reused as the "anchor" id so `PersistentAnnotationStore`
  can key and upsert the pAnn stably; the `affectorId` override carries the same value)

Strategy: in the per-permanent loop, also accumulate morbid cards grouped by player. After the
loop, for each `perPlayer` condition, check the condition against a representative card from
that player's morbid permanents; if true, emit one entry.

```kotlin
// Per-player accumulation: conditionName → (seatId → list of iids)
val perPlayerCards = mutableMapOf<String, MutableMap<Int, MutableList<Int>>>()

for (card in battlefieldCards) {
    val controller = card.controller ?: continue
    val iid = instanceIdResolver(ForgeCardId(card.id)).value

    // ... existing per-permanent scan (Phase 1 + Phase 2) ...
    // After emitting per-permanent entries, also check for perPlayer conditions:

    // Phase 3: accumulate perPlayer morbid candidates
    for (sa in card.staticAbilities ?: emptyList()) {
        val condition = sa.getParam("Condition") ?: continue
        val spec = CONDITIONS[condition] ?: continue
        if (!spec.perPlayer) continue
        val seatId = controller.seatOf()  // see note below
        perPlayerCards
            .getOrPut(condition) { mutableMapOf() }
            .getOrPut(seatId) { mutableListOf() }
            .add(iid)
    }
    for (trigger in card.triggers ?: emptyList()) {
        for (paramName in NAMED_PARAM_CONDITIONS) {
            val spec = CONDITIONS[paramName] ?: continue
            if (!spec.perPlayer) continue
            if (!trigger.hasParam(paramName)) continue
            val seatId = controller.seatOf()
            perPlayerCards
                .getOrPut(paramName) { mutableMapOf() }
                .getOrPut(seatId) { mutableListOf() }
                .add(iid)
        }
    }
}

// Emit one pAnn per player per perPlayer condition (only when condition is true)
for ((conditionName, bySeat) in perPlayerCards) {
    val spec = CONDITIONS[conditionName] ?: continue
    for ((seatId, iids) in bySeat) {
        // Pick any card as the Forge `src` param (for game reference)
        val anyCard = battlefieldCards.firstOrNull { it.controller?.seatOf() == seatId } ?: continue
        val conditionValue = spec.value?.invoke(anyCard.controller!!, anyCard) ?: 1
        if (conditionValue <= 0) continue  // condition not met — omit pAnn

        results.add(
            AbilityWordEntry(
                instanceId = seatId,           // stable key for PersistentAnnotationStore upsert
                abilityWordName = conditionName,
                value = if (spec.booleanOnly) null else conditionValue,
                threshold = if (spec.booleanOnly) null else spec.threshold,
                affectorId = seatId,
                affectedIds = iids,            // all morbid permanents for this player
            ),
        )
    }
}
```

Note on `controller.seatOf()`: check if Player has a seat number accessor in this Forge version.
If not, derive it from `bridge` — but the scanner is pure (no bridge param). Alternative: use
`controller.ordinal + 1` or compare against `game.registeredPlayers` ordering. Verify in code
before implementing; adjust if needed.

**Also update `AbilityWordEntry`:** add `affectedIds: List<Int> = emptyList()` field. The existing
per-permanent entries leave it empty; per-player entries populate it. `StateMapper` must be
updated to pass this through to `AnnotationBuilder.abilityWordActive()`.

Current `AnnotationBuilder.abilityWordActive()` signature adds only one `affectedIds` entry
(`addAffectedIds(instanceId)`). For per-player entries we need all iids. Either add an
`extraAffectedIds: List<Int>` param to the builder call, or — simpler — add an overload:

```kotlin
fun abilityWordActive(
    instanceId: Int,
    abilityWordName: String,
    value: Int? = null,
    threshold: Int? = null,
    abilityGrpId: Int? = null,
    affectorId: Int = instanceId,
    affectedIds: List<Int> = listOf(instanceId),  // NEW: default = single iid (existing behavior)
): AnnotationInfo = AnnotationInfo.newBuilder()
    .addType(AnnotationType.AbilityWordActive)
    .setAffectorId(affectorId)
    .apply { affectedIds.forEach { addAffectedIds(it) } }  // iterate list
    .addDetails(typedStringDetail(DetailKeys.ABILITY_WORD_NAME, abilityWordName))
    .apply {
        if (value != null) addDetails(int32Detail(DetailKeys.VALUE, value))
        if (threshold != null) addDetails(int32Detail(DetailKeys.THRESHOLD, threshold))
        if (abilityGrpId != null) addDetails(int32Detail(DetailKeys.ABILITY_GRP_ID_UPPER, abilityGrpId))
    }
    .build()
```

Default `affectedIds = listOf(instanceId)` preserves all existing callers unchanged.

Update `StateMapper.kt` scan-to-annotation map block to pass `entry.affectedIds`:

```kotlin
AnnotationBuilder.abilityWordActive(
    instanceId = entry.instanceId,
    abilityWordName = entry.abilityWordName,
    value = entry.value,
    threshold = entry.threshold,
    abilityGrpId = entry.abilityGrpId,
    affectorId = entry.affectorId ?: entry.instanceId,
    affectedIds = entry.affectedIds.ifEmpty { listOf(entry.instanceId) },
)
```

test("Morbid card with dead creature emits per-player AbilityWordEntry") {
    val (b, game, _) = base.startWithBoard { _, human, ai ->
        base.addCard("Cackling Prowler", human, ZoneType.Battlefield)
        // Simulate a creature death: add a Creature card to the GY with origin=Battlefield
        // Use game.action.moveToGraveyard or addCard then manipulate cardsAddedThisTurn
        base.addCard("Grizzly Bears", ai, ZoneType.Battlefield)
    }
    val human = game.humanPlayer
    val prowler = human.getZone(ZoneType.Battlefield).cards.first { it.name == "Cackling Prowler" }

    // Kill the AI creature to trigger morbid condition
    val ai = game.players.find { it != human }!!
    val bear = ai.getZone(ZoneType.Battlefield).cards.first()
    game.action.moveToGraveyard(bear, null)

    val results = AbilityWordScanner.scan(
        battlefieldCards = game.registeredPlayers.flatMap { it.getZone(ZoneType.Battlefield).cards.toList() },
        instanceIdResolver = { fid -> b.getOrAllocInstanceId(fid) },
        registryResolver = { card ->
            val grpId = b.cards.findGrpIdByName(card.name) ?: 0
            b.abilityRegistryFor(card, b.cards.findByGrpId(grpId))
        },
    )

    val morbidEntry = results.firstOrNull { it.abilityWordName == "Morbid" }
    morbidEntry.shouldNotBeNull()
    val prowlerIid = b.getOrAllocInstanceId(ForgeCardId(prowler.id)).value
    morbidEntry.affectorId shouldBe /* human seatId */ 1
    morbidEntry.affectedIds shouldContain prowlerIid
    morbidEntry.value shouldBe null        // boolean-only
    morbidEntry.threshold shouldBe null    // boolean-only
}

test("Morbid card with no creature death emits no entry") {
    val (b, game, _) = base.startWithBoard { _, human, _ ->
        base.addCard("Cackling Prowler", human, ZoneType.Battlefield)
    }

    val results = AbilityWordScanner.scan(
        battlefieldCards = game.humanPlayer.getZone(ZoneType.Battlefield).cards.toList(),
        instanceIdResolver = { fid -> b.getOrAllocInstanceId(fid) },
        registryResolver = { _ -> null },
    )

    results.filter { it.abilityWordName == "Morbid" }.shouldBeEmpty()
}

test("Morbid and Threshold on same scan does not interfere") {
    val (b, game, _) = base.startWithBoard { _, human, _ ->
        base.addCard("Cackling Prowler", human, ZoneType.Battlefield)
        base.addCard("Dreadwing Scavenger", human, ZoneType.Battlefield)
        repeat(5) { base.addCard("Plains", human, ZoneType.Graveyard) }
    }
    val human = game.humanPlayer
    val bear = base.addCard("Grizzly Bears", human, ZoneType.Battlefield)
    game.action.moveToGraveyard(bear, null)

    val results = AbilityWordScanner.scan(
        battlefieldCards = human.getZone(ZoneType.Battlefield).cards.toList(),
        instanceIdResolver = { fid -> b.getOrAllocInstanceId(fid) },
        registryResolver = { card ->
            val grpId = b.cards.findGrpIdByName(card.name) ?: 0
            b.abilityRegistryFor(card, b.cards.findByGrpId(grpId))
        },
    )

    results.count { it.abilityWordName == "Morbid" } shouldBe 1
    results.count { it.abilityWordName == "Threshold" } shouldBe 1
}

test("Morbid pAnn has seatId affectorId and all morbid permanents in affectedIds") {
    val (b, game, counter) = base.startWithBoard { _, human, ai ->
        base.addCard("Cackling Prowler", human, ZoneType.Battlefield)
        base.addCard("Grizzly Bears", ai, ZoneType.Battlefield)
    }
    val human = game.humanPlayer
    val prowler = human.getZone(ZoneType.Battlefield).cards.first { it.name == "Cackling Prowler" }
    val prowlerIid = b.getOrAllocInstanceId(ForgeCardId(prowler.id)).value

    // Kill AI bear — triggers morbid condition
    val bear = game.players.find { it != human }!!.getZone(ZoneType.Battlefield).cards.first()
    val gsm = base.captureAfterAction(b, game, counter) {
        game.action.moveToGraveyard(bear, null)
    }

    val morbidAnns = gsm.persistentAnnotationsList.filter {
        AnnotationType.AbilityWordActive in it.typeList &&
            it.detailString("AbilityWordName") == "Morbid"
    }
    morbidAnns shouldHaveSize 1
    assertSoftly {
        morbidAnns[0].affectorId shouldBe 1                     // P1 seatId
        morbidAnns[0].affectedIdsList shouldContain prowlerIid   // Cackling Prowler iid
        morbidAnns[0].detailString("AbilityWordName") shouldBe "Morbid"
        // Boolean-only: no value or threshold details
        morbidAnns[0].detail("value") shouldBe null
        morbidAnns[0].detail("threshold") shouldBe null
    }
}

test("Morbid pAnn absent when no creature died") {
    val (b, game, counter) = base.startWithBoard { _, human, _ ->
        base.addCard("Cackling Prowler", human, ZoneType.Battlefield)
    }

    val gsm = base.stateOnlyDiff(game, b, counter)
    gsm.persistentAnnotationsList.filter {
        AnnotationType.AbilityWordActive in it.typeList &&
            it.detailString("AbilityWordName") == "Morbid"
    }.shouldBeEmpty()
}

test("Morbid pAnn cleared on next diff after turn resets") {
    val (b, game, counter) = base.startWithBoard { _, human, ai ->

    }
    val human = game.humanPlayer
    val bear = game.players.find { it != human }!!.getZone(ZoneType.Battlefield).cards.first()

    // First diff — morbid active
    val gsm1 = base.captureAfterAction(b, game, counter) {
        game.action.moveToGraveyard(bear, null)

    gsm1.persistentAnnotationsList.filter {
        AnnotationType.AbilityWordActive in it.typeList &&
            it.detailString("AbilityWordName") == "Morbid"

    // Simulate turn boundary — reset thisTurnEntered tracking

    val gsm2 = base.captureAfterAction(b, game, counter) {}
    gsm2.persistentAnnotationsList.filter {
        AnnotationType.AbilityWordActive in it.typeList &&
            it.detailString("AbilityWordName") == "Morbid"
    }.shouldBeEmpty()
}

test("Two morbid cards produce single pAnn with both iids in affectedIds") {
    val (b, game, counter) = base.startWithBoard { _, human, ai ->
        base.addCard("Cackling Prowler", human, ZoneType.Battlefield)
